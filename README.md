# irembo-system-design-interview

## Run Frontend

```bash
$ cd client
$ npm install
$ npm run dev
```

---
## Run Backend

```bash
cd user-account-management
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
